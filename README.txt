#Pasos para instalar el proyecto
##Requisitos
- Node.js >= v16
- NPM

Clonar el repositorio:
    git clone []
Instalar dependencias o verificar que existen:
    npm install
Ejecutar Cypress:
    npm run cypress:open

#Pasos para ejecutar Flujo de Compra E2E en Saucedemo
En la interfaz gráfica de Cypress:
    1. Seleccionar E2e Testing
    2. Seleccionar Chrome
    3. Seleccionar Botón 'Start E2E Testing in Chrome'
    4. En la ventana Specs, seleccionar purchase.cy.js


#Pasos para ejecutar pruebas de API [PetStore] con Cypress.io

En la interfaz gráfica de Cypress:
    1. Seleccionar E2e Testing
    2. Seleccionar Chrome
    3. Seleccionar Botón 'Start E2E Testing in Chrome'
    4. En la ventana Specs, seleccionar petstore.cy.js
