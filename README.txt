#Pasos para instalar el proyecto
##Requisitos
- Node.js >= v16
- NPM

Clonar el repositorio:
    git clone https://github.com/Ozymandias-II/AlexanderVargasTest-.git
Instalar dependencias o verificar que existen:
    npm install
Ejecutar Cypress:
    npm run cypress:open
